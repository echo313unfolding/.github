## Projects

### Compressed AI Stack

Three repos form a complete compressed AI runtime:

| Repo | What | Key Result |
|------|------|------------|
| **[helix-substrate](https://github.com/echo313unfolding/helix-substrate)** | Weight compression | 4x, beats GPTQ quality, calibration-free |
| **[helix-online-kv](https://github.com/echo313unfolding/helix-online-kv)** | KV cache compression | 1.9x more context, 87.5% attention sparsity |
| **[echo_runtime](https://github.com/echo313unfolding/echo_runtime)** | Unified runtime | All three layers in one forward pass |

The codec is the same everywhere: VQ-256 codebook + uint8 indices + outlier sidecar. Works on transformers, SSMs, CNNs, vision encoders, and embedding models without architecture-specific code.

### Forensic Graph Intelligence

| Repo | What | Key Result |
|------|------|------------|
| **[FGIP](https://github.com/echo313unfolding/FGIP)** | Graph intelligence platform | 1,801 nodes, 3,286 edges, adversarial-tested theses |

Tracks lobbying, ownership, and judicial networks from government data (SEC EDGAR, FEC, Congress.gov, Federal Register, FRED). Every thesis is adversarial-tested with control groups.

## Methodology

- **Receipts or it didn't happen** -- every claim backed by verifiable JSON receipts with cost blocks
- **Adversarial-first** -- find the strongest counter-argument before claiming verification
- **Real data only** -- synthetic test data is flagged, not trusted
- **Proof layers**: (1) code runs, (2) correct output format, (3) real data, (4) statistical significance
